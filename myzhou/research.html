Exploration of IoT service security
Investigated the potential security problem in the authentication between the cloud and third-party servers. Validated the attack model through proof-of-concept experiments on Google Home. Found out an erroneous implementation of the OAuth protocol on Google Home.

Analysis of Swarm users' share-to-Twitter behavior
Investigated the factors that impact Swarm users' behavior of sharing their check-ins to Twitter. Used statistical analysis and machine learning algorithms to evaluate the influence of factors. The results can help researchers identify whether their Twitter-collected check-ins can represent the user behavior on Swarm.
Paper An Empirical Study of the Usage of the Swarm App's Cross-Site Sharing Feature
Workshop AppLens, UbiComp 2018

WeChat Mini Program: LBSLab
Developed the entire server of a Wechat mini-program, LBSLab, which allows users to make check-ins, record emotions, and check the weather. It also serves as a user data collection system for researchers. The number of registered users has reached 2510 by Nov. 2018.
Paper LBSLab: A User Data Collection System in Mobile Environments
Workshop MHC, UbiComp 2018

Measurement of the 4G network on high-speed rails
Measured the performance of TCP with CUBIC, TCP with BBR, MPTCP in the high-speed environment. Developed a tool to split the mixed TCP packets into independent TCP flows. Wrote Python scripts to match the data packets collected from mobile phones with those from computers and servers. Used BladeRF to collect data of the link layer in order to find out the reason of performance changes.

Analysis of Swarm users' behavior in cities
Analyzed the user behavior through spatial and temporal features of Swarm check-ins. The results reveal the activity patterns of human behavior and the distinctions of life habits among three cities, i.e. Hong Kong, New York City, and San Francisco.
Paper Understanding User Activity Patterns of the Swarm App: A Data-Driven Study
UbiComp 2017, Poster Session

Analysis of Skout users' mobility patterns
Crawled the data of posts created by more than 1.2 million users on Skout, a location based social network. Constructed a global city network through Skout posts to figure out users' mobility patterns among cities.
Paper Understanding Skout Users’ Mobility Patterns on a Global Scale: A Data-Driven Study
World Wide Web Journal (WWWJ), 2018
